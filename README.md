# 📚 Hindi Sahitya Book Store

A modern React-based book store web application where users can browse, search, and filter books, and admin can add new books dynamically.

---

## 🚀 Features

✅ Beautiful UI with responsive design  
✅ Book listing with images  
✅ Search functionality  
✅ Category filtering  
✅ Price range filter  
✅ Admin panel to add books  
✅ Image upload with preview  
✅ LocalStorage integration (data persists after refresh)

---

## 🛠️ Tech Stack

- React JS ⚛️
- CSS3 🎨
- JavaScript (ES6)
- LocalStorage (for saving books)

## ➕ How to Add Book (Admin Panel)

1. Go to Admin page  
2. Fill:
   - Title  
   - Author  
   - Price  
   - Upload Image  
3. Click **Add Book**

👉 Book will:
- Save in localStorage  
- Show on homepage automatically
